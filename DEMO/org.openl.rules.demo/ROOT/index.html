OpenL Tablets DEMO

Welcome to OpenL Tablets DEMO

OpenL Tablets Demo contains configured and ready to demonstrate infrastructure for OpenL Tablets product. The Demo enables a user to develop business rules and manage business rule sets in OpenL Tablets WebStudio, and then execute these rules as Web Services ready to be used by solution applications.
WebStudio WebStudio is a web application employed by business users and developers to view, edit, and manage business rules and rule projects created using OpenL Tablets technology. In order to make work with WebStudio more comfortable and quick to start, several rule projects are already loaded in Repository. Production repository is configured the same way in both WebStudio and Web Services applications. A project from Template - "Example 3 - Auto Policy Calculation" is pre deployed there.
Web Services Web Services application provides the ability to dynamically create web services for customer rules offering extensive configuration and customization capabilities. Rules of the deployed project "Example 3 - Auto Policy Calculation" are exposed as web services and called in Web Services Client application.
Web Services Demo Client Web Services Demo Client is an application specially created to demonstrate usage of OpenL Tablets rules via Web Services application. Web Services Demo Client is a very simple application and should be used just to check that rules are easily accessible from other applications via web services. Rules of preloaded and deployed project "Example 3 - Auto Policy Calculation" are invoked to define driver type by his age and gender, get premium per accident or determine how risky a driver.
