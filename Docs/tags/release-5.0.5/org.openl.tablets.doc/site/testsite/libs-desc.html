Here is the (almost)complete list of OpenL Tablets Dependencies

We have to admit it is rather big list. Fortunately, most of the dependencies are UI dependencies - runtime dependencies are still being kept to the reasonable minimum. See deployment document for runtime dependencies. OpenL Tablets also uses slightly modified version Apache POI library - version 2.5.1. We provide it together with the source code. OpenL Tablets thanks all the open-source contributors for their hard work!
