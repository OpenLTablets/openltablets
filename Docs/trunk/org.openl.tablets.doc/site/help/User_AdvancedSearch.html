Advanced Search

Advanced search allows the user to narrow the search by specifying criteria for tables where the search must be performed.

To perform an advanced search, proceed as follows:

  1. Open the search window as described in Performing a Search.

  2. To switch to the advanced search mode, click Advanced Search.

    3. The advanced search window appears.

  3. In the Table Selector section, enter search criteria for tables in which the search must be performed.

  4. In the Column Selector section, enter search criteria for columns in which the search must be performed.

  5. Click Search.

    6. OpenL Web Studio displays tables matching the search criteria below the advanced search window. The user can switch to table editing mode directly from the advanced search window by hovering the mouse button over a table and selecting Edit. For information on modifying tables, see Modifying Tables.
