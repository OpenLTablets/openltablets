OpenL Tablets Webstudio User's Guide



Preface

This document is intended for OpenL Tablets developers and business users who plan to use Webstudio to develop, debug and maintain OpenL Tablets projects. For developers the familiarity with Java, Eclipse and MS Excel are assumed. Business users need to be familiar with basic web browser navigation and MS Excel editing/formatting. No knowledge of VBA or Excel macros is required.

Webstudio Layout

The Webstudio allows users to browse OpenL Project structure, select Project Components, edit Project Components directly in Excel; in addition it can Run and Trace the Executable Components. The results of Run are displayed in the browser and are available to inspect.
