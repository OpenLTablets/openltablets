OpenL Tablets Web Studio Installation

Web Studio prototype is currently available only as Eclipse project from OpenL Tablets CVS.

1. Install OpenL Tablets Plugin version 3.1.0.7 Build 20 or higher

2. In Eclipse create a Repository Location using the following credentials:

:pserver:anonymous@openl-tablets.cvs.sourceforge.net:/cvsroot/openl-tablets

3. Checkout the following projects from the folder /PROTOTYPE:

  • org.openl.lib.apache.tomcat
  • org.openl.rules.webtools
  • org.openl.rules.webstudio
    •

    Running Web Studio

    1. In Eclipse Run menu locate command StartTomcat Webstudio

    2. Point your browser to http://localhost:8080/webstudio/

    3. OpenL Tablets Web Studio was developed under Mozilla Firefox and somewhat tested with IE Explorer. We used Office 2000 for office-related testing. In case if you have problems using Web Studio with other configurations (unlikely) please report them to the project forums.

    See also OpenL Tablets Web Studio Manual
