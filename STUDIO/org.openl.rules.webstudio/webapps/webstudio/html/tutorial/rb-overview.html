Repository Browser Overview

The Repository Browser allows users to browse repository, edit projects.

This document demonstrates and briefly explains how to use OpenL Tablets Repository Browser.

General View

Repository Browser General View consists of the two parts: Repository Explorer and Project Component View

Repository Explorer

The Repository Explorer provides tree-like view that allows users to browse projects and project elements.

    Project states

Project can be in one of the following states:

  • Checked out
  • Opened
  • Closed
  • Deleted
  • Local
  • Opened, locked

    Project elements

  • Folder
  • Document
