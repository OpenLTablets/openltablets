Select user mode:

User mode defines how many users can run the application and where user projects will be located by default.

Single-user – Only the currently logged in user can run OpenL Tablets WebStudio. All user projects will be saved in the root of the '#{installWizard.workingDir}/user-workspace' directory.

Multi-user – Multiple users can run OpenL Tablets WebStudio using their unique user names. The User’s projects will be located in the '#{installWizard.workingDir}/user-workspace/USERNAME' folder.

Select application mode:

Here you should choose which Database to use for managing users in the application.

Production – You will use an external MySQL database to manage OpenL Tablets WebStudio users. The database is customizable allowing you to add/remove users and change their access rights.

Demo – You will use an internal User Database based on the HSQLDB database engine with predefined list of users. All changes in the database will be lost after the application restart.

Configure database:

Set up the MySQL Database to be used for managing users in OpenL Tablets WebStudio.
Please contact your System Administrator for this information if necessary.

Click 'Finish' to complete the installation process
